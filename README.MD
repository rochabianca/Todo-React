# ToDo App React
Um pequeno WebApp de cadastro de tarefas feito para aprofundar os conhecimentos na biblioteca React

## Requisitos

* **Mongodb**
* **NodeJS**

## Instalação

* **Backend:**

```
npm install
```

```
mongod
```

```
npm run dev
```

* **Frontend**

```
npm install
```
```
npm run dev
```
